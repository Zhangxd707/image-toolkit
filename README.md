# Image Processing Toolkit

A lightweight command-line toolkit for basic image processing, built with NumPy and Pillow for learning purposes.

## Features

- ** Grayscale Conversion **: Convert color images to 8-bit grayscale
- ** Brightness Adjustment **: Scale pixel values with configurable factor
- ** Otsu Thresholding **: Automatic optimal thresholding for binary images
- ** Median Filtering **: Remove salt-and-pepper noise with variable kernel size
